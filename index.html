physicsarena

Link to project in development - Link to final project

>Type: Website

>Languages: HTML, CSS, Javascript

>Language: English (and Romanian)

>Time: February-April 2021

I decided to recreate fizicarea with a more expandable design. I also wanted to remake the interface so it is also easy to expand and also make it a liitle bit more modern.

It is a bit of a hassle, because I decided not to use a framework like React. I think that some websites that use these frameworks take a lot of time to load and I wanted to see if I could make a website that is not dependent of a system built by somebody else.

If I want to make another web app in the future, it is almost guaranteed that I will make it in React.

Johnson Game

Download executable - Link to repository

>Type: Windows App

>Version: 0.2

>Languages: C++ (with SFML)

>Language: Romanian

>Time: July-August 2020

This is a 2D sandbox game I made using C++ and a lot od dedication.

It is more like an engine for games in a similar style to Undertale or Pokemon, so it can be repurposed.

Once in a while I compile the executable, so you can just download the game, unzip the contents and play the game.

This project was done with the help of my friend, who designed the graphics.
